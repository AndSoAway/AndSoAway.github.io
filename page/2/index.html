Serialize and Deserialize Binary Tree

最常见的是用BFS，用＃来表示NULL。

但是，实际上用DFS是最方便的。 to_string(root->val) + “,” + serialize(root->left) + “,” + serialize(root->right).
然后deserialize的时候，首先得到的是root的val，然后递归得到左子树的，右子树的。因为碰到＃，就可以直接返回NULL了。所以不会陷入不能分解的地步。
